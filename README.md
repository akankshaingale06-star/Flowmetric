Features:
- Login and Register pages (frontend simulation)
- Academic To-Do List with priority levels
- Coming deadlines and monthly overview
- Subject-wise color coding
- Formula vault for important equations
- Code snippet section for labs and projects
- Professor-wise task tracking
- Lab work management
- Achievement badges for motivation
- Clean and calming UI (blue, pastel green, white)

- Tech Stack"
- HTML
- CSS
- JavaScript (Frontend only)
- No backend or database required

-How to run the project:
To run the project in VS Code, install Visual Studio Code and open the project folder containing Flowmetric.html. Install the Live Server extension from the Extensions tab, then open index.html, right-click inside the file, and select Open with Live Server. The website will open in your browser and automatically refresh when you make changes.
